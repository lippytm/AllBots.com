# AllBots.com

**AI Full-Stack DevOps & Blockchain Learning Platform**

An embedded GitHub application for AI Synthetic Intelligence Engines, AgentBots, and Swarms. AllBots.com teaches and trains people, robotics, and bots to become better programmers and blockchain technology developers — and lets you **earn while you learn**.

---

## Features

| Feature | Description |
|---|---|
| 🤖 **Bot Dashboard** | Deploy, monitor, and orchestrate AI AgentBots and Swarms |
| 🎓 **Learning Platform** | Full-stack, DevOps, Blockchain, AI, Robotics, and Security tracks |
| 💰 **Earn While You Learn** | Earn AllBots Tokens (ABT) for every module completed, bug fixed, or course contributed |
| 🐙 **GitHub App Integration** | Automated code reviews, CI/CD orchestration, smart-contract auditing, and PR bots |

---

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router, TypeScript)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Linting**: ESLint with `eslint-config-next`

---

## Getting Started

### Prerequisites

- Node.js 18+
- npm 9+

### Install & Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm run start
```

### Lint

```bash
npm run lint
```

## GitHub App Integration

AllBots.com is designed to be installed as a GitHub App in your organization or personal account. Once installed, AI agents can:

- Review pull requests automatically
- Run CI/CD pipelines via bots
- Audit Solidity smart contracts on every push
- Propose dependency and refactoring PRs
- Sync learning achievements to your GitHub profile
