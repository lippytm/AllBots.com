# AllBots.com

Bots managements and automations with imbedded GitHub.

## OpenRouter Multi-Model Routing

This repo now includes an OpenRouter routing prototype so AllBots can choose the best model for each bot task instead of locking every workflow to one provider.

- Guide: `OPENROUTER_ROUTING.md`
- Router: `bots/openrouter_model_router.py`
- Configure with `OPENROUTER_API_KEY` and optional task-specific model environment variables.

Money angle: use AllBots to package task-specific AI automations for businesses, then route startup/business leads to `https://lippytmai.getbizfunds.com` and cloud-computer operations to `lippytmai.zo.computer`.
